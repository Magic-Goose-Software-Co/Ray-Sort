
# Ray

A simple to use email sorter named after [Ray Tomlinson](https://en.wikipedia.org/wiki/Ray_Tomlinson).

---


## Setup

Download the repo
```bash
git clone https://github.com/magic-goose-software-co/ray-sort.git
```

Run setup.py (You don't have to `cd` into it)
```bash
python ray/setup.py
```
and answer its five questions.

---

## Usage

Move some emails into the mailboxes you want it to sort mail into and run `python ray`.
